# Outfit Recommendation API

## Project Description
This project is a backend API that helps users generate outfit combinations from the clothes they own.

Users can:
- Add clothing items to their wardrobe
- View and manage their clothing items
- Generate outfit combinations
- Save outfits they like

## Tech Stack
- Python
- Django
- Django REST Framework
- PostgreSQL (later)
